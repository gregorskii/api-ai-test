# API.AI Test Project

Example project providing quick access to an API.ai project over via a NodeJS API.

# Prerequisites

- NodeJS ~ 6.9.1

# Setup

Install packages using:

```
npm install
```

Create a `.env` file with the content:

```
API_AI_CLIENT_KEY=###
```

Replace `###` with your accounts Client API key.

Run:

```
node index.js
```
